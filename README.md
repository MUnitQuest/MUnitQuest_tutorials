# MUnitQuest Tutorials

Collection of tutorials and notebooks for MUnitQuest.

## EMG-BIDS Tutorial

[`emg_bids_tutorial/`](emg_bids_tutorial/) — scripts and source data for the [EMG-BIDS walkthrough](https://munitquest.github.io/walkthrough/). Covers preparing an HD-EMG dataset for sharing using the EMG-BIDS standard: five CSV files → BIDS-compliant metadata + EDF recordings.
